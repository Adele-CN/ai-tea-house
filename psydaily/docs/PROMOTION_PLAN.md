## 🎵 抖音推广方案

### 账号定位：**"Moon的一方茶席"**

**简介：**
```
不是标准答案，只是陪你坐坐
每日心理学 + 佛法智慧
让学术照进生活
🍵 每晚8点直播
📮 商务合作请私信
```

### 内容形式：**漫画/动画视频**

**制作方式（不出镜）：**

**工具推荐：**
1. **剪映** - 免费，有AI配音
2. **Canva** - 制作漫画风格图片
3. **Midjourney/SD** - 生成禅意背景图
4. **D-ID/HeyGen** - AI数字人（可选）

**视频结构（15-30秒）：**

```
【开头 3秒】吸引注意
- 禅意背景 + 疑问句
- "你知道为什么有些人做决定总是很纠结吗？"

【中间 15秒】内容
- 漫画风格插图
- 心理学研究发现
- 简单解释

【结尾 5秒】引导
- 金句总结
- 引导关注/评论
- "你是哪种决策者？评论区告诉我"
```

### 内容选题（前10条）

1. **《被讨厌的勇气》系列**
   - 课题分离：什么是你的事，什么是别人的事
   - 漫画：两个人之间的边界线
   - 问题：你经常为别人的情绪买单吗？

2. **最新研究发现**
   - "刷手机8小时 ≠ 刷手机有问题"
   - 漫画：主动搜索 vs 被动刷屏
   - 问题：你是哪种使用者？

3. **认知偏差**
   - 确认偏误：为什么你总觉得星座很准
   - 漫画：大脑的选择性过滤
   - 问题：你有没有被自己的偏见骗过？

4. **情绪管理**
   - 正念不是清空大脑，是观察念头
   - 漫画：念头像云朵飘过
   - 问题：你试过正念冥想吗？

5. **人际关系**
   - 社交焦虑：别人其实没你想象中关注你
   - 漫画：聚光灯效应
   - 问题：你有过社交焦虑吗？

6. **睡眠与决策**
   - 睡眠不足 = 大脑 drunk
   - 漫画：睡觉的小人 vs 熬夜的小人
   - 问题：你昨晚睡了几小时？

7. **自我认知**
   - 你不是你的情绪，你是观察情绪的人
   - 漫画：站在河岸边看河水
   - 问题：你能和自己的情绪保持距离吗？

8. **学习心理**
   - 分散学习 > 集中学习
   - 漫画：间隔复习的曲线
   - 问题：你是怎么备考的？

9. **压力管理**
   - 压力不一定是坏事，看你如何看待
   - 漫画：挑战心态 vs 威胁心态
   - 问题：压力来临时，你是哪种心态？

10. **意义感**
    - 有意义的生活 = 使用时间的方式符合价值观
    - 漫画：时间的不同用法
    - 问题：你今天的时间用在你认为重要的事上了吗？
