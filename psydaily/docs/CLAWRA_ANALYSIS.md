# Clawra 分析报告

## 这是什么？

**Clawra** = OpenClaw 的 AI 女友

### 核心功能
1. **AI 自拍生成** - 用 fal.ai 生成 consistent 的自拍照
2. **跨平台发送** - 支持 Discord/Telegram/WhatsApp 等
3. **场景响应** - 用户说"你在干嘛？""发张照片"时回复图片

### 技术栈
- 图像生成：xAI Grok Imagine (via fal.ai)
- 参考图固定：CDN 托管的固定形象
- 消息发送：OpenClaw Gateway

---

## 我们可以做的：心理学陪伴宠物

### 概念：**PsyPet** (心理学陪伴精灵)

**不是** AI 女友（主打颜值/情感陪伴）
**而是** 心理学专业陪伴（主打认知支持+情绪调节）

### 差异化定位

| Clawra (AI女友) | PsyPet (心理学陪伴) |
|----------------|---------------------|
| 发自拍、聊情感 | 发"心理状态图"、给认知工具 |
| 外观吸引 | 内涵有用 |
| 被动陪伴 | 主动干预 |
| 颜值经济 | 实用价值 |

### 核心功能

**1. 心理状态可视化**
- 用户说"我现在很焦虑"
- PsyPet 生成一张图：小怪物被信息漩涡包围
- 同时给出 3 分钟缓解方案

**2. 认知工具交付**
- 用户说"帮我清空大脑"
- PsyPet 生成"大脑清理完成"的庆祝图
- 附带今天完成的心理练习

**3. 每日心理打卡**
- 早上：发送"今日心理能量条"
- 晚上：发送"今日认知消耗图"

**4. 心理学知识卡片**
- 用图片形式发送心理学概念
- 比纯文字更生动

### 技术实现

参考 Clawra 的架构：

```
PsyPet Skill
├── 图像生成 (fal.ai/Midjourney API)
├── 固定参考图 (形象一致)
├── 场景触发器
│   ├── "我很焦虑" → 焦虑缓解图+方案
│   ├── "帮我冥想" → 冥想引导图+音频
│   ├── "今天状态" → 心理能量可视化
│   └── "学到了" → 知识卡片图
└── 跨平台消息发送
```
